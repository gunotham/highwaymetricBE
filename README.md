# Highwaymetric Backend

This is the backend service for the Highwaymetric application. It is built using Java and Spring Boot.

## Getting Started

### Prerequisites

*   Java 17 or higher
*   Maven 3.6 or higher

### Building the Project

To build the project, navigate to the root directory and run:

```bash
mvn clean install
```

### Running the Application

To run the application, use the following command:

```bash
mvn spring-boot:run
```

The application will typically run on `http://localhost:8080`.

## Project Structure

*   `src/main/java`: Contains the main Java source code.
*   `src/main/resources`: Contains application resources like `application.properties`.
*   `src/test/java`: Contains unit and integration tests.
